# EDGP Phase 2.5: 前端 UI 设计实施计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 为 EDGP 静态博客系统设计并实现一套美观、简洁、有科技感的前端 UI。

**Architecture:** 基于 CSS 变量构建统一的设计系统，通过重构 `site_templates/` 模板和编写新的 CSS 文件实现全站 UI 升级。采用移动端优先的响应式设计。

**Tech Stack:** CSS3 (Custom Properties), Bootstrap 5 (本地化), HTML5 语义化标签，Flexbox/Grid 布局
