# EDGP Phase 1.5: 本地入口页重构

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 将本地端首页从"完整博客页"重构为"极简入口页"，仅提供跳转到 Admin 的入口

**Architecture:** 新增 `local_templates/` 目录存放本地端模板，`site_templates/` 保留为静态生成专用

**Tech Stack:** Django Templates, Bootstrap 5 (已存在)

---

## 背景

**问题**: Phase 1 实现了完整的博客首页模板，但用户反馈：
> "本地撰写、生成页面，在线仓库展示页面" — 本地端不需要"好看"的网站页面

**解决方案**: 
1. 本地端改为极简入口页
2. site_templates/ 作为静态生成专用模板

## Task 2: 更新 settings.py 模板配置

**Files:**
- Modify: `edgp/settings.py`

**Step 1: 添加 LOCAL_TEMPLATES_DIR 配置**

在 settings.py 中找到 TEMPLATES 配置，修改 DIRS：

```python
# 原配置
TEMPLATES = [
    {
        'BACKEND': 'django.template.backends.django.DjangoTemplates',
        'DIRS': [BASE_DIR / 'site_templates'],  # 修改这里
        ...
    },
]

# 修改为
TEMPLATES = [
    {
        'BACKEND': 'django.template.backends.django.DjangoTemplates',
        'DIRS': [
            BASE_DIR / 'local_templates',   # 本地端模板优先
            BASE_DIR / 'site_templates',    # 静态生成模板
        ],
        ...
    },
]
```

**Step 2: 添加自定义设置**

在文件末尾添加：

```python
# 模板目录配置
LOCAL_TEMPLATES_DIR = BASE_DIR / 'local_templates'
SITE_TEMPLATES_DIR = BASE_DIR / 'site_templates'
```

**Step 3: Commit**

```bash
git add edgp/settings.py
git commit -m "feat: configure local_templates in settings"
```
